# 📸 Instagram Clone

## 📖 Project Overview

This project is a **front-end clone of Instagram** built using **HTML, CSS, and JavaScript**.
It replicates the basic user interface and layout of Instagram to demonstrate **modern web development concepts such as responsive design, UI structuring, and interactive elements**.

The project was developed as a **practice project to improve front-end development skills and understand how popular social media interfaces are structured.**

---

## 🚀 Features

* Instagram-like **homepage layout**
* **Responsive design** for different screen sizes
* Interactive **navigation bar**
* **Post feed interface**
* **Like and interaction UI**
* Clean and modern **user interface**

---

## 🛠 Technologies Used

* **HTML5** – Structure of the web pages
* **CSS3** – Styling and responsive design
* **JavaScript** – Basic interactivity and dynamic behavior

---

## ⚙️ How to Run the Project

1. Clone the repository

```
git clone https://github.com/YOURUSERNAME/instagram-clone.git
```

2. Open the project folder

3. Run the project by opening:

```
index.html
```

in your web browser.
